# 支持

## 获取帮助

请优先通过 GitHub Issue 提问或报告问题。提交前建议先搜索已有 Issue，避免重复讨论。

可复现 bug 请使用 GitHub Bug 报告表单，并参考 `docs/bug-reporting.md` 准备最小复现。安全问题请参考 `SECURITY.md`，不要在公开 Issue 中披露未修复漏洞细节。

## 提问时请提供

- 使用的项目版本或提交哈希。
- JDK 版本和操作系统。
- 执行的 Gradle 命令。
- 期望结果和实际结果。
- 最小复现代码、配置或测试用例。
- 完整异常堆栈或构建日志中最相关的片段。

## 适合提交 Issue 的内容

- 可复现的 bug。
- 文档缺失或说明不清。
- 构建、测试、发布流程问题。
- 兼容性问题。
- 经过说明的功能请求。

## 不适合公开 Issue 的内容

- 未修复安全漏洞的详细利用步骤。
- 私有凭据、内部地址、生产数据或用户隐私。
- 与本项目无关的通用 LDAP 运维问题。
