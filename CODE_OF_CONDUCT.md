# 行为准则

[English version](docs/en/CODE_OF_CONDUCT.md)

本项目希望为所有参与者提供开放、友好、专业的协作环境。

## 我们鼓励

- 尊重不同经验水平、背景和观点。
- 聚焦事实、代码、设计和用户影响。
- 给出可操作反馈，说明原因和取舍。
- 在意见不一致时保持耐心，优先寻找可验证的结论。
- 主动帮助新贡献者理解项目结构和流程。

## 我们不接受

- 人身攻击、骚扰、侮辱、歧视或威胁。
- 发布他人隐私信息、凭据或未授权资料。
- 持续刷屏、挑衅、偏离主题或阻碍协作。
- 在安全问题未修复前公开漏洞细节。
- 明知会破坏项目、用户或维护者安全的行为。

## 执行方式

维护者可以对违反行为准则的内容进行提醒、编辑、隐藏、关闭讨论、限制参与或移交平台处理。处理时会尽量兼顾上下文、影响范围和改正意愿。

## 报告问题

如果你遇到或观察到不当行为，请联系项目维护者。报告中请尽量提供链接、时间、相关人员和简要经过。
