# 1.0.0-M21 发布说明

发布日期：2026-06-23

## 概览

这是仓库迁移为纯 Gradle 多模块项目后的首个发布准备版本。该版本集中补齐 Gradle 7.6 构建、版本属性管理、开源项目入口文档、中央仓库发布任务和中文 DN 非 ASCII 大小写转换修复。

## 主要变化

- 项目迁移为 Gradle 7.6 多模块构建，不再维护 Maven POM。
- 项目版本和依赖版本统一维护在 `gradle.properties`。
- 补齐 Maven Central/OSSRH 发布任务，支持本地签名、`publishToMavenLocal` 门禁和 staging 上传。
- 补齐 README、构建、架构、发布准备、贡献、安全、支持、行为准则、Bug 报告和英文副本文档。
- 修复 `Strings.toLowerCaseAscii` 处理中文 DN 等非 ASCII 字符时可能抛出数组越界异常的问题。
- 修复 Gradle 构建下 OSGi 测试依赖装配、manifest 合并和 bundle 名称适配问题。

## 兼容性说明

- 发布坐标版本为 `1.0.0-M21`。
- 构建入口切换为 Gradle Wrapper；不再使用 Maven reactor。
- Java 源码仍按 Java 8 兼容性编译。

## 发布门禁

发布前需要运行：

```powershell
.\gradlew.bat releaseGate
```

该任务会执行完整构建、OSGi 测试和 `publishToMavenLocal`，并验证 POM、sources jar、javadoc jar 和签名产物可生成。
