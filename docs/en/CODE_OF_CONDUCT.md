# Code of Conduct

[中文版本](../../CODE_OF_CONDUCT.md)

This project aims to provide an open, welcoming, and professional collaboration environment for all participants.

## We Encourage

- Respect for different experience levels, backgrounds, and viewpoints.
- Discussion focused on facts, code, design, and user impact.
- Actionable feedback that explains reasons and trade-offs.
- Patience during disagreement, with a preference for verifiable conclusions.
- Helping new contributors understand the project structure and workflow.

## We Do Not Accept

- Personal attacks, harassment, insults, discrimination, or threats.
- Publishing another person's private information, credentials, or unauthorized materials.
- Repeated spam, provocation, off-topic disruption, or obstruction of collaboration.
- Public disclosure of vulnerability details before a security issue is fixed.
- Behavior that knowingly harms project, user, or maintainer safety.

## Enforcement

Maintainers may warn about, edit, hide, or close problematic content, limit participation, or escalate to platform-level handling. Enforcement will consider context, impact, and willingness to correct the behavior.

## Reporting

If you experience or observe inappropriate behavior, contact the project maintainers. Please include links, timestamps, people involved, and a brief description when possible.
