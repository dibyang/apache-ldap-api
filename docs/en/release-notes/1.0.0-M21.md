# 1.0.0-M21 Release Notes

Release date: 2026-06-23

## Overview

This is the first release-preparation version after the repository was migrated to a Gradle-only multi-module project. It focuses on the Gradle 7.6 build, centralized version properties, open source project entry-point documentation, Maven Central publishing tasks, and a fix for non-ASCII case conversion with Chinese DN values.

## Highlights

- Migrated the project to a Gradle 7.6 multi-module build and removed Maven POM maintenance.
- Centralized project and dependency versions in `gradle.properties`.
- Added Maven Central/OSSRH publishing tasks with local signing, a `publishToMavenLocal` gate, and staging upload support.
- Added README, build, architecture, release preparation, contributing, security, support, code of conduct, bug reporting, and English translation documents.
- Fixed a possible array index error in `Strings.toLowerCaseAscii` when handling Chinese DN values and other non-ASCII characters.
- Fixed Gradle-based OSGi test dependency assembly, manifest merging, and bundle name adaptation.

## Compatibility Notes

- Published coordinates use version `1.0.0-M21`.
- The build entry point is now the Gradle Wrapper; the Maven reactor is no longer used.
- Java source remains compatible with Java 8.

## Release Gate

Before publishing, run:

```powershell
.\gradlew.bat releaseGate
```

This task runs the full build, OSGi tests, and `publishToMavenLocal`, and verifies that POMs, sources jars, javadocs jars, and signed artifacts can be generated.
