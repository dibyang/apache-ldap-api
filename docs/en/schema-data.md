# Schema Data

[中文版本](../../ldap/schema/data/README.md)

This document is the English companion for the `ldap/schema/data` module README.

The Chinese primary document records the schema OID allocation policy generated by `/src/main/scripts/oid_allocation.pl`. OIDs, attribute type names, object class names, and schema names are protocol and data identifiers, so they are intentionally kept in their original spelling and case.

See the Chinese primary document for the full generated OID allocation list.
